# 项目的第一阶段设计文档
## 项目架构
项目的架构采用的标准的Controller,Service,Dao分层的架构，在项目的第一阶段，Controller,Service和Dao
层同时在一台电脑上运行


## 项目技术栈
预计使用的技术栈为Hertz+GRPC+Gorm+Redis+Kafka+MySQL\
在项目的第一阶段，主要是应用到了Hertz,Gorm,Mysql,Redis暂时设置为第二部分的操作
